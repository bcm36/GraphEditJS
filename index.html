GraphEditJS

A javascript-based property graph editor.
Inspired by rkirsling.

About

This is very much still under development: breaking changes are guaranteed.

Installation

Dependencies

This plugin relies on D3, Bootstrap and jQuery.

To build from the coffeescript source, compile with:

coffee -c coffee/graphedit.coffee js/graphedit.js

Import the needed CSS resources:

<head>
...
<link href="css/graphedit.css" rel="stylesheet">
<link href="css/bootstrap.min.css" rel="stylesheet">
</head>

And the required JS resources:

<body>
...
<script src="js/jquery.min.js"></script>
<script src="js/d3.v3.js"></script>
<script src="js/graphedit.js"></script>
</body>

Then invoke as described in Usage.

Usage

via JavaScript

Create an instance of GraphEdit via javascript:

$('.graphedit').graphEdit();

Data format

All nodes should be specified with this format:

{node_id: <node id>}

And all edges should be specified with this format:

{src: <node id>, dest: <node id>}

Options

Name type default description
nodes list of objects [] Sets the nodes in the graph. See Data format for more info.
edges list of objects [] Sets the edges in the graph. See Data format for more info.

Methods

.graphEdit(options)

Activates a GraphEdit editor. Accepts an optional options object.

$("#myGraphEdit").graphEdit();

.graphEdit('addNode', node)

Adds the node represented by object node

$("#myGraphEdit").graphEdit('addNode', {node_id: 7, name:"Alice", age: 25});

.graphEdit('addEdge', edge)

Adds the node represented by object edge

$("#myGraphEdit").graphEdit('addEdge', {src: 7, dest: 10, label:"knows"});
