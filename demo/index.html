GraphEditJS

A javascript-based graph editor that supports property editing.
Inspired by rkirsling.

Usage

via JavaScript

Create an instance of GraphEdit via javascript:

$('.graphedit').graphEdit({});

Data format

All nodes should be specified with this format:

{node_id: node id}

And all edges should be specified with this format:

{src: node id, dest: node id}
